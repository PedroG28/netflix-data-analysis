# Netflix Data Analysis

## Description
This project analyzes the Netflix catalog using SQL and Power BI.

## Tools
- PostgreSQL
- SQL
- Power BI

## Dataset
Netflix Movies and TV Shows dataset.

## Analysis
The analysis answers questions such as:

- Movies vs TV Shows distribution
- Catalog growth over time
- Top countries producing Netflix content
- Most common ratings

## Dashboard
The dashboard visualizes Netflix catalog trends.
